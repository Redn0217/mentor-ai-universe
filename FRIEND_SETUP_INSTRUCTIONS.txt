--------------------------------------------------------------------------------
TROUBLESHOOTING
--------------------------------------------------------------------------------

If you still get "supabaseUrl is required":
1. Make sure backend\.env exists
2. Open it and verify it has the SUPABASE_URL line
3. Restart the server (Ctrl+C and run npm run dev:all again)

If port 8080 is busy:
- Vite will automatically use 8081 instead (that's fine!)

If nothing works:
1. Delete node_modules folders
2. Run: npm cache clean --force
3. Run: npm install
4. Try again
